# **App Name**: Course Compass

## Core Features:

- Semester Columns: Display eight columns, one for each semester, to organize courses.
- Course Input: Allow users to add course information to each semester column.
- Course Rearrangement: Enable drag-and-drop functionality to rearrange courses within and between semesters.
- Intuitive Interface: Provide a clean and intuitive interface for easy navigation and course management.
- AI Course Suggestion: Use an AI tool to suggest related courses based on the courses already added. This can help students discover relevant electives or specializations.
- Theme Selection: Allow users to switch between a dark and a light theme via a button.
- Mouse Reactive Animation: Create cool vector animation in the background that reacts to the mouse.
- User Authentication: Allow users to register and login using their Gmail accounts without email confirmation.

## Style Guidelines:

- Primary color: Teal (#008080) for a professional and academic feel in light mode.
- Secondary color: Light gray (#F0F0F0) for backgrounds and subtle contrasts in light mode.
- Accent: Purple (#800080) to highlight important information and interactive elements in light mode.
- Primary color: Dark Gray (#333333) for a sleek and modern feel in dark mode.
- Secondary color: Medium Gray (#666666) for backgrounds and subtle contrasts in dark mode.
- Accent: Light Purple (#A95AA9) to highlight important information and interactive elements in dark mode.
- Clean and organized layout with clear separation between semesters.
- Use academic-themed icons for visual cues and easy identification of course types.
- Subtle animations for drag-and-drop actions to provide visual feedback.
- Cool vector animations in the background that react to mouse movement and clicks.
